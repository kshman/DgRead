# 득읽기(DgRead)

득읽기는 Avalonia를 이용해서 만든 *이미지* 또는 *책*을 보는 프로그램입니다. (그니깐 **만화뷰어** 😣)

## 무언가 설명
전에 만든 [큭북][1]과 [두뷰][2]를 바탕으로 AI를 어떻게 해야 잘 썼나 검토하기 위해 **Avalonia UI**의 **80%** 이상 *코파일럿*에게 시켜서 만들거나 기존 두뷰의 내용을 변환했습니다.
그렇게 해서 만들어 봤는데, AI가 다 잘하는건 아니더군요. 빼먹고, 구형 로직을 쓰고, 하라는대로 안하고 말이죠. 그래도 80%까진 하는것도 대단합니다.

논리 전개와 코딩은 그다지 별로였는데, 감동받은거는 *프로파일링*은 쩌네요. 로딩 문제로 고민하다 이야기를 던져봤는데, 
알아서 프로파일링을 하면서 벤치마크 프로젝트도 생성하고 도와주더라구요. 그래서 큭북과 두뷰에 비해서 GIF/WEBP **애니메이션 로딩**이 엄청 빨라졌습니다.

아무튼 부족한 기능도 있고 생각해둔 기능도 있어서 더 해봐야겠어요. 계속 진행해볼께요.

# 사용 가이드

여기에 설명한 사용 가이드는 모두 AI가 만든 사용법입니다.

## 1) 시작하기

- 책/파일 열기: `Ctrl+O`
- 폴더 열기: `Ctrl+Shift+O`
- 창에 파일/폴더를 드래그 앤 드롭해도 열 수 있습니다.

## 2) 기본 페이지 이동

- 다음: `Space`, `NumPad0`, `Right`
- 이전: `Left`
- 처음/끝: `Home`, `End`
- 페이지 점프(설정 단위, 기본 10쪽): `PageDown`(앞), `PageUp`(뒤), `Back`(앞)
- 1쪽 이동: `,`(이전), `.` 또는 `/`(다음)

> 참고: `Ctrl+PageUp/PageDown`은 이전/다음 책 열기입니다.

## 3) 보기 모드 전환

- 단일 보기: `1`
- 좌→우 두 페이지 보기: `2`
- 우→좌 두 페이지 보기: `3`
- 세로 스크롤 보기: `` ` ``
- 좌/우 보기 방향 토글: `Tab`

## 4) 확대/축소

- 즉시 확대/축소 모드 진입: `F8`
- 확대: `+`
- 축소: `-`
- 휠 확대/축소: 확대 모드일 때, 또는 `Shift`+휠, 또는 우클릭 상태+휠
- 확대 상태 이동(팬): 방향키
- 확대 해제/맞춤: `/`(상황에 따라 페이지 넘김 키와 겹칠 수 있음)

## 5) 스크롤 모드 조작

세로 스크롤 보기에서:
- 마우스 드래그로 스크롤
- 방향키로 스크롤
- `PageUp/PageDown`으로 화면 단위 이동
- `+`/`-`로 폭 비율 조정

## 6) 책/파일 작업

- 현재 책 닫기: `Ctrl+W`
- 이전/다음 책: `[` / `]` (또는 브라우저 뒤/앞 키)
- 랜덤 책 열기: `\` 또는 `|`
- 이동: `Insert`
- 이름 변경: `F2`
- 삭제: `Delete`
- 마지막 파일 다시 열기: `Ctrl+Shift+Z`

## 7) 책갈피

- 책갈피 추가: `F9` 또는 `P`
- 책갈피 관리 창: `F10` 또는 `Ctrl+P`
- 이미 같은 위치 책갈피가 있으면 삭제할지 확인 창이 뜹니다.

## 8) 창 제어

- 전체화면 토글: `F` 또는 `Alt+Enter`
- `Esc`:
  - 전체화면이면 전체화면 해제
  - 설정에서 `Esc 종료`가 켜져 있으면 프로그램 종료
