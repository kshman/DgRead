# 득읽기(DgRead)

득읽기는 Avalonia를 이용해서 만든 *이미지* 또는 *책*을 보는 프로그램입니다. (그니깐 **만화뷰어** 😣)

전에 만든 [큭북][1]과 [두뷰][2]를 바탕으로 AI를 어떻게 해야 잘 썼나 검토하기 위해 **Avalonia UI**의 **80%** 이상 *코파일럿*에게 시켜서 만들거나 기존 두뷰의 내용을 변환했습니다.
그렇게 해서 만들어 봤는데, AI가 다 잘하는건 아니더군요. 빼먹고, 구형 로직을 쓰고, 하라는대로 안하고 말이죠. 그래도 80%까진 하는것도 대단합니다.

논리 전개와 코딩은 그다지 별로였는데, 감동받은거는 *프로파일링*은 쩌네요. 로딩 문제로 고민하다 이야기를 던져봤는데, 
알아서 프로파일링을 하면서 벤치마크 프로젝트도 생성하고 도와주더라구요. 그래서 큭북과 두뷰에 비해서 GIF/WEBP **애니메이션 로딩**이 엄청 빨라졌습니다.

아무튼 부족한 기능도 있고 생각해둔 기능도 있어서 더 해봐야겠어요. 계속 진행해볼께요.
